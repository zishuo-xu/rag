# 行业评测语料说明

本目录中的文档是为了 RAG 评测而整理的结构化样本，不是原始官方全文转存。

当前语料覆盖三类主题：

1. 智驾方案与平台
   - 伯特利智能底盘与海外布局样本
   - 地平线 HSD 量产智驾样本
   - 小鹏 XNGP 全国都好用样本
   - 百度 Apollo 量产智驾与云服务样本
   - 蔚来 NAD 与 NOP+ 样本
   - 比亚迪天神之眼与整车智能样本

2. 自动驾驶公司商业化
   - 文远知行产品矩阵与商业化样本
   - 小马智行全球化业务样本

3. 政策与基础设施
   - 智能网联汽车准入与上路通行试点样本
   - 车路云一体化应用试点样本

整理原则：

- 每份文档都按章节写成短结构，便于稳定切 chunk 和做章节级评测。
- 文档内容以公开资料、官方页面和本仓库已有行业总结为基础，做了评测导向的压缩整理。
- 文档优先保留“定义、定位、能力、商业化、政策目标、关键部门、区域信息”等适合做问答评测的事实。
